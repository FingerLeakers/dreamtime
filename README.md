## Setup

> All the source code is inside the src/ folder, from now on all the instructions will be based on this folder.

### Quick Setup

Install dependencies:

`yarn install`

Start development enviroment:

`yarn start:nuxt`

`yarn start:babel`

`yarn start:electron`

Build:

`yarn build`
